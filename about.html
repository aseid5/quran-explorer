What is Quran Explorer?

Quran Explorer is a web-based application that allows users to search for verses from the Quran using keywords. It connects to the Al-Quran Cloud API to return results that match the searched term.

Why this project was built

The purpose of this project was to make Quranic content easier to explore. It was designed for users who want to find and reflect on verses based on specific themes or topics.

How it works

On the Explore page, users can type in a keyword. The app then fetches and displays verses that include that keyword. This feature helps users navigate the Quran in a more interactive and topic-focused way.
